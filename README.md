# Bookcase-system
这款微信小程序书橱系统，打造私人云端书库。打开小程序，扫码就能录入书籍信息，为每本藏书建立数字档案，生成专属书单。支持标签分类，无论是文学名著、专业教材，还是科普读物，都能轻松归类。书橱搜索功能强大，通过书名、作者或关键词，快速定位想找的书。还能与好友分享藏书，推荐优质书籍，发起线上读书交流，在书海共享阅读乐趣 。让你随时随地管理书籍，畅享阅读生活。
